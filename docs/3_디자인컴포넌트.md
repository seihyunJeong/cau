### 1-3. 간격 시스템 (Spacing)

4px 그리드 기반. 모든 간격은 4의 배수로 통일한다.

| 토큰 이름 | 값 (dp) | 용도 |
|---|---|---|
| `xxs` | 2 | 아이콘과 텍스트 사이 미세 간격 |
| `xs` | 4 | 인라인 요소 사이, BottomSheet 리스트 아이템 내부 |
| `sm` | 8 | 카드 내 요소 사이, 칩 사이, 아이콘-텍스트 간격 |
| `md` | 12 | 카드 내부 패딩 (compact), 섹션 내 요소 간격 |
| `base` | 16 | 기본 수평 패딩, 카드 내부 패딩 (standard), 리스트 아이템 간격 |
| `lg` | 24 | 섹션 사이 간격, 카드 사이 간격, 화면 상단/하단 여백 |
| `xl` | 32 | 주요 섹션 사이 간격, 온보딩 화면 상단 여백 |
| `xxl` | 48 | 화면 최상단 여백 (세이프 에리어 아래), 온보딩 일러스트 주변 여백 |

#### 화면 레이아웃 간격

| 위치 | 값 (dp) | 비고 |
|---|---|---|
| 화면 좌우 수평 패딩 | 16 | `EdgeInsets.symmetric(horizontal: 16)` |
| 카드 내부 패딩 | 16 | `EdgeInsets.all(16)` |
| 카드 내부 패딩 (컴팩트) | 12 | 프로필 카드, 빠른 기록 버튼 |
| 카드 사이 수직 간격 | 12~16 | `SizedBox(height: 12)` 또는 `SizedBox(height: 16)` |
| 섹션 타이틀과 내용 사이 | 8~12 | |
| 섹션 사이 간격 | 24 | |
| 하단 고정 버튼 내부 패딩 | 16 수평, 12 수직 | `EdgeInsets.fromLTRB(16, 12, 16, 16)` + SafeArea |
| AppBar 높이 | 56 | Flutter 기본값 사용 |
| 하단 탭 바 높이 | 56~64 | `BottomNavigationBar` 기본값 |

| 레벨 | 엘리베이션 | 용도 | BoxShadow 정의 |

#### 2-1-1. 하단 탭 바 (Bottom Tab Bar / 하단 네비게이션)

**사용 화면:** 홈, 기록, 활동, 발달, 마이 (전체 메인 화면)

**시각 스펙:**

- 활성 탭 색상: `#F5A623` (warmOrange) - 아이콘 + 레이블 모두
- 비활성 탭 색상: `#8C7B6B` (warmGray)

##### InfoTerm 용어 데이터 (개발기획서 1-4 기반, 총 9개)

아래 테이블의 모든 용어는 `InfoTerm` 위젯으로 앱 내 해당 텍스트가 등장하는 모든 곳에 적용한다.

| # | termName (전문 용어) | parentLabel (앱 화면 표현) | description (Tooltip 설명) |
|---|---|---|---|
| 1 | 모로반사 (Moro Reflex) | 깜짝 놀라며 팔을 벌리는 반응 | 갑작스러운 소리나 움직임에 놀라 양팔을 벌렸다가 모으는 자연스러운 반응이에요. 보통 3~4개월에 자연스럽게 사라져요. |
| 2 | 탐색반사 (Rooting Reflex) | 입 주변을 건드리면 고개를 돌리는 반응 | 볼이나 입 주변을 건드리면 그쪽으로 고개를 돌리는 반응이에요. 수유와 연결되는 중요한 반응이에요. |
| 3 | 파악반사 (Palmar Grasp) | 손바닥을 건드리면 꽉 쥐는 반응 | 손바닥에 무언가 닿으면 자동으로 꽉 쥐는 반응이에요. 나중에 물건 잡기의 기초가 돼요. |
| 4 | 각성 (Arousal) | 깨어 있는 시간 | 아기가 잠에서 깨어 주변을 인식하는 상태예요. 신생아는 하루 중 깨어 있는 시간이 매우 짧아요. |
| 5 | 공동조절 (Co-regulation) | 부모가 안아주며 함께 안정되는 것 | 아기가 스스로 진정하기 어려울 때, 부모의 안기·목소리·리듬으로 함께 안정되는 과정이에요. |
| 6 | 과자극 (Overstimulation) | 자극이 너무 많아 지친 상태 | 빛, 소리, 움직임 등 자극이 아기가 처리할 수 있는 양을 넘어선 상태예요. 울음이나 고개 돌림으로 나타나요. |
| 7 | 굴곡 자세 (Flexion Posture) | 팔다리를 몸 쪽으로 모은 자세 | 자궁 안에서 익숙했던 웅크린 자세예요. 신생아가 편안할 때 자연스럽게 이 자세를 취해요. |
| 8 | 신경계 통합 (Neural Integration) | 아기의 뇌와 몸이 안정을 배우는 과정 | 아기의 뇌가 감각 정보를 받아들이고 처리하는 방법을 익혀가는 과정이에요. |
| 9 | consolability | 달래주면 진정되는 정도 | 울거나 불편해할 때 보호자가 안아주면 얼마나 빨리 안정되는지를 나타내요. |

**트리거:** 타이머가 0:00에 도달했을 때 (무음 모드 활성 상태)
**동작:** 전체 화면 오버레이, white → transparent 불투명도 펄스
**지속 시간:** 200ms x 2회 펄스 (총 400ms)
**이징:** `Curves.easeInOut`
**부가 효과:** `HapticFeedback.mediumImpact()`와 함께 실행

```dart
// 화면 전체를 덮는 오버레이로 구현
// AnimatedOpacity 또는 ColorFiltered overlay 활용
Stack(
  children: [
    // 기존 타이머 화면
    timerContent,
    // 플래시 오버레이
    if (_isFlashing)
      AnimatedOpacity(
        opacity: _flashOpacity,  // 1.0 → 0.0 → 1.0 → 0.0 (2회 펄스)
        duration: Duration(milliseconds: 200),
        curve: Curves.easeInOut,
        child: Container(color: Colors.white),
      ),
  ],
)

// 트리거 시 haptic 동반
HapticFeedback.mediumImpact();
```
